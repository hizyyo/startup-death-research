# Case Studies

# The Anatomy of Startup Death

## Overview

This directory contains deep-dive case studies from the 100-case MVP dataset.

Each case study is a mini research paper analyzing one startup failure in detail.

## Selection Criteria

Cases were selected for:

- strong primary-source evidence
- clear failure mechanism
- diverse failure causes
- diverse categories and geographies
